# ABAP CDS 内置数据类型

ABAP CDS 中的内置数据类型基于 ABAP 字典的内置类型，并遵循与相应字典类型相同的规则。ABAP CDS 中的内置数据类型以域前缀 `abap`、句点 (.) 及数据类型名称组成。

以下表格列出了所有可用的内置 CDS 数据类型及其可映射到的对应字典类型。第三列指示该数据类型是否可用于 CDS 元素的类型定义。第四列指示是否可用于 CDS 参数的类型定义。第五列指示是否可在 CAST 表达式中使用。

| 表示法                          | ABAP 字典中的内置数据类型           | 元素类型定义 | 参数类型定义 | CAST 目标 |
| ------------------------------- | ----------------------------------- | ------------ | ------------ | --------- |
| `abap.char( len )`              | CHAR，长度为 `len`                  | x            | x            | x         |
| `abap.clnt[(3)]`                | CLNT                                | x            | x            | x         |
| `abap.cuky[(5)]`                | CUKY                                | x            | x            | x         |
| `abap.curr(len,dec)`            | CURR，长度为 `len`，小数位为 `dec`  | x            | x            | x         |
| `abap.d16n[(16)]`               | DECFLOAT16                          | x            | x            | x         |
| `abap.d34n[(34)]`               | DECFLOAT34                          | x            | x            | x         |
| `abap.datn[(8)]`                | DATN                                | x            | x            | x         |
| `abap.dats[(8)]`                | DATS                                | x            | x            | x         |
| `abap.dec(len,dec)`             | DEC，长度为 `len`，小数位为 `dec`   | x            | x            | x         |
| `abap.fltp[(16,16)]`            | FLTP                                | x            | x            | x         |
| `abap.geom_ewkb`                | GEOM_EWKB                           | x            | -            | -         |
| `abap.int1[(3)]`                | INT1                                | x            | x            | x         |
| `abap.int2[(5)]`                | INT2                                | x            | x            | x         |
| `abap.int4[(10)]`               | INT4                                | x            | x            | x         |
| `abap.int8[(19)]`               | INT8                                | x            | x            | x         |
| `abap.lang[(1)]`                | LANG                                | x            | x            | x         |
| `abap.numc( len )`              | NUMC，长度为 `len`                  | x            | x            | x         |
| `abap.quan(len,dec)`            | QUAN，长度为 `len`，小数位为 `dec`  | x            | x            | x         |
| `abap.raw(len)`                 | RAW，长度为 `len`                   | x            | x            | -         |
| `abap.rawstring`                | RAWSTRING                           | x            | -            | -         |
| `abap.sstring(len)`             | SSTRING，长度为 `len`               | x            | x            | x         |
| `abap.string`                   | STRING                              | x            | -            | -         |
| `abap.timn[(6)]`                | TIMN                                | x            | x            | x         |
| `abap.tims[(6)]`                | TIMS                                | x            | x            | x         |
| `abap.unit[(2|3)]`              | UNIT，长度为 2 或 3（标准长度）     | x            | x            | x         |
| `abap.utcl[(27)]`               | UTCLONG                             | x            | x            | x         |

## 说明

- **`len` 和 `dec`**：必须用于指定相应通用类型的长度和小数位数。此处括号中指定的值必须在 ABAP 字典允许的值范围内。
- **固定长度和小数位数的数据类型**：可以（但不是必须）指定预定义的长度和小数位数值。
- **例外情况 `abap.accp`**：此类型具有预定义的长度 6，**必须**始终在括号中指定。

## 在 CDS 简单类型中的使用

除了 `abap.accp` 和 `abap.fltp` 之外，上表列出的所有数据类型均可用于类型定义。
- 数据类型 `abap.accp` 已过时。
- 可以使用十进制浮点数类型来代替 `abap.fltp`。

## 在 CDS 表实体中的使用

除了 `abap.accp`、`abap.clnt`、`abap.dats`、`abap.tims` 和 `abap.geom_ewkb` 之外，上表列出的所有数据类型均可用于 CDS 表实体的类型定义。
