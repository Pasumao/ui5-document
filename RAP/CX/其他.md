

## @ObjectModel.semanticKey: []
> 可以通过使用注解 @ObjectModel.semanticKey 来定义语义关键字段，该注解包含实体中的一个或多个属性数组。所指定的字段将以粗体显示，并且还会显示其编辑状态。
>

cds
```
@ObjectModel.semanticKey: ['Name']
```

## @Metadata.ignorePropagatedAnnotations: true
> 定义了在使用注释 API CL_DD_DDL_ANNOTATION_SERVICE 对 CDS 对象进行分析时，是否继承来自已使用的 CDS 对象的
>

cds
```
@AccessControl.authorizationCheck: #NOT_REQUIRED
@EndUserText.label: 'YCX_VH_TRAVELID'
@Metadata.ignorePropagatedAnnotations: true
define root view entity YCX_VH_TRAVELID as select from ycxtestdata001
{
    ...
}

```

## @UI.multiLineText
> 通过添加 @UI.multiLineText 标注，较长的字符串将显示在多行中。

cds
```
@UI.multiLineText: true
Description;

@UI.multiLineText: true 
DescriptionCustomGrowing;
```
